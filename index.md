## Sobre as representações inspiradas no Portugol

Portugol é uma pseudolinguagem de programação em português, projetada para ensinar lógica de programação de forma acessível. Os exemplos neste site e em meu livro seguem uma sintaxe inspirada no Portugol. Abaixo temos um exemplo básico da sintaxe utilizada.

```portugol
Algoritmo: "Exemplo básico"

Entrada Inteiro numero

Inicio
    Leia numero
    
    Se numero > 0 Entao
        Escreva "O número é positivo"
    Senao
        Escreva "O número é zero ou negativo"
    FimSe
Fim
```
